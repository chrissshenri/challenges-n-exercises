## 📁 How to practice as well?

```bash

    # Clone the repository
    $ git clone https://github.com/chrissshenri/challenges-n-exercises

    # Navigate to the directory
    $ cd challenges-n-exercises

    # Install the dependencies
    $ yarn install

    # Start the project
    $ yarn start

```
