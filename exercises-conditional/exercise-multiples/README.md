## 👀 About the problem

Create a program to read two integer numbers and determine if one number is a multiple of the other. The numbers can be entered in any order.

---

## 📁 How to practice as well?

```bash

    # Clone the repository
    $ git clone https://github.com/chrissshenri/challenges-n-exercises-js.git

    # Navigate to the directory
    $ cd challenges-n-exercises-js

    # Install the dependencies
    $ yarn install

    # Start the project
    $ yarn start

```
