## 👀 About the problem

Create a program that reads N integer numbers and stores them in an array. Then, display on the screen all the even numbers and also the count of even numbers.

---

## 📁 How to practice as well?

```bash

    # Clone the repository
    $ git clone https://github.com/chrissshenri/challenges-n-exercises-js.git

    # Navigate to the directory
    $ cd challenges-n-exercises-js

    # Install the dependencies
    $ yarn install

    # Start the project
    $ yarn start

```
