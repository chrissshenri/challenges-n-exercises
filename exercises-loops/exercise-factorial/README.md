## 👀 About the problem

Create a program to read a natural number N (maximum value: 15), and then calculate and display the factorial of N.

---

## 📁 How to practice as well?

```bash

    # Clone the repository
    $ git clone https://github.com/chrissshenri/challenges-n-exercises-js.git

    # Navigate to the directory
    $ cd challenges-n-exercises-js

    # Install the dependencies
    $ yarn install

    # Start the project
    $ yarn start

```
